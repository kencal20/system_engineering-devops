This project is about learning shell variables, environment variables, and expansions in Linux.
It covers how to create and use aliases, define local and global variables, and modify important environment variables like PATH.
It also shows how to perform arithmetic operations using shell expansions and environment variables.
The goal is to understand how the shell manages variables and how they can be used to customize behavior and improve scripts.
This is a beginner project to practice working with variables and expansions in the Linux shell.
