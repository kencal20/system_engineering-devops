# 0x04-loops_conditions_and_parsing

This project focuses on Bash scripting fundamentals including loops, conditional statements, and file parsing techniques.

## 📌 Topics Covered
- for loops
- while loops
- until loops
- if / elif / else statements
- case statements
- file testing operators
- string manipulation in Bash

## 📁 Project Structure

Each script demonstrates a specific concept:

- `1-for_best_school` → for loop basics
- `2-while_best_school` → while loop usage
- `3-until_best_school` → until loop logic
- `4-if_9_say_hi` → conditional logic inside loops
- `5-4_bad_luck_8_is_your_chance` → if/elif/else branching
- `6-superstitious_numbers` → case statement usage
- `8-for_ls` → filename parsing using for loops
- `9-to_file_or_not_to_file` → file existence and type checks
